# Diary Create — 다이어리 채널 자동 생성 스킬

티켓 채널에서 사용자가 "다이어리 생성"을 요청하면 이 스킬을 호출한다. 모든 단계를 결정적으로 실행한다.

## 트리거

티켓 카테고리 하위 채널에서 다음과 같은 메시지가 오면:
- "다이어리 생성"
- "다이어리 만들어주세요"
- "다이어리 만들어줘"
- 비슷한 다이어리 생성 요청

## 사용법

```bash
bash /home/node/.claude/skills/diary-create/create-diary.sh <user_id> <ticket_channel_id>
```

- `user_id`: 요청자의 Discord 사용자 ID (메시지 sender_id)
- `ticket_channel_id`: 현재 티켓 채널 ID

## 동작

8단계 자동 실행:

1. 티켓 채널에 "잠시만 기다려주세요! 🦉" 즉시 발송
2. Discord API로 사용자 정보 조회 (역할 ID, 닉네임)
3. 사용자의 기숙사 역할 매칭 (4개 기숙사 중)
4. 닉네임에서 `[직장인]`, `[대학생]` 등 대괄호 부분 제거
5. 해당 기숙사 카테고리에 다이어리 채널 생성
6. 사용자에게 권한 오버라이드 (View+Send+ManageMessages, Deny ManageChannels)
7. 다이어리 채널에 사용자 멘션
8. 티켓 채널에 완료 메시지 + 채널 링크

## 에러 처리

- 사용자 정보를 못 가져오면 → "사용자 정보를 가져올 수 없어요" 메시지 후 종료
- 기숙사 역할이 없으면 → "기숙사 역할이 없어서 다이어리를 만들 수 없어요" 메시지 후 종료
- 채널 생성 실패 → "채널 생성에 실패했어요" 메시지 후 종료
